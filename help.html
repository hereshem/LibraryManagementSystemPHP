About This Manual

This manual is created in order to provide information for every user about how to use the Library Management System.

About the Library

This Project was created for the use of library in the Purwanchal Engineering Campus.

Overview of this Program

This Program is very helpful in the library as it does the following function :
  • Total Up To Date Records of the Transaction done in the library.
  • Keeping all the details of each book in the library.
  • Keeping the details of all the student.
  • Can get information of the transactions according to the date, Book history by whom it was taken before And the record of each student book transaction during the period.
  • Simple E-mail facilities and Other facilities.

Operating System Requirements

This section lists the operating systems on which you can expect to be able to run the Library Management system.
This system is specially designed with WAMP.So obviously it supports the windows operating system version such as windows 98,XP,Proffessional,windows server 2000,etc.

Third Party Softwares Requirement

Besides the operating system, this programs needs other third party softwares also. WAMP packages softwares such as Apache 2.0 , Mysql server 5.0 And PHP(Hypertext Preprocessor)3.0 or the Precombined package such as TRIAD .

Installation Layout

This section describes how to install the system in the computer.

First of all, the windows operaing system and the WAMP package (Apache, MYSQL and PHP) must be installed. Now COPY the library Folder to the clipboard and PASTE it to the Web Folder(i.e C:/WWW folder or C:/apache/htdocs folder).

Connecting the WAMP

To connect the WAMP, first run the 'controlpanel.exe' in C:/apache. The windows Appears

Click on "Install Apache as Service", Click "back" then again Click on "Start Apache", Click back and then Click on "Start MySQL".
Now type "http://localhost" on the address field of the Browser. The front page appears.

Creating Databases And Tables In Database

Type "http//:localhost/library" in the address field of browser. The front Page Appears. Type User Name and Password and click on Submit button. If the Database is not connected then the message appears To click the link for creating the database and tables. Now you can sign in and enjoy the site.

Creating The Users with Different Privileges

For the smooth running and security purpose Different types of users and the servers must be created.
Here four types of servers can be created with their own privileges.

  • ADMINISTRATOR
    The Server with administrator rights have all the privileges such as book entry, student entry, issue and return books, create and delete the users, change the students details, Reset Passwords, etc. The Administrator has got all these rights.
  • ISSUE RIGHT

    Those servers with Issue Rights have the privileges for only issuing the books from the library.
  • RETURN RIGHT

    The server with Returning Rights can only accept the books that is returned by the student.


  • And the Student Right have the access to entry the NEW STUDENTS.

    • Entering The Student Details

    Entering the student detail is very easy. The Administrator has the right to enter the detail of students.
    OR the user with student right also have the privileges to enter the student detail. Every student can enter their detail themselves.

    The requirements for entering the student details are :
    • A Photo with passport size in CD or Pendrive.
    • Personal Details
    • And the PERSONAL PASSWORD, which is required while issuing books or getting other facilities such as E-mail.

    Entering The Books

    Entering the book can only be done by the Administrator.

    Issue And Returning Books Transactions

    • Book issue transaction can be done by the particular user with the grant of ISSUE RIGHT along with the personal password of the student by whom it is taken.

    • Returning of the book is performed by the particular person with the grant of RETURN RIGHT.

    Specifying Program Option

    The Program Consists of the following functions :
  • Search For Books
    This option is used for searching the books.
  • Issue Books
    This option is enabled only for the necessary privilaged person which is used for issuing the books along with the personal password of student.
  • Return Books
    This option is used to submit the book to the library and get the card free.
  • Entry Books
    This option is used to enter the new books in the library and get the Accession Number of the book. this option is enabled to the Administrator Only
  • New Students
    This option is also used to enter the details of the student and password. This option is enabled to the person with student right or Administrator. To get the User Name and Password for entering the Student Detail Please Contact To the Administrator.
  • Search Students
    You can Search any student with the very few details or with no details provided to it. You can Search student according to the Batch, Faculty, Roll Number, First Name, Last Name or Address.
  • Today's History
    With this option you can View All the Books Transaction done on That day (TODAY) or any other day with the Date
  • Books History
    This option is Used to Know the history of the book by which student that book was taken By Providing the Accession Number of that Book.
  • Syllabus
    This option is used to view all the Syllabus according to the Faculty wise. The Syllabus for the IE entrance and BE entrance is also given in the page.
  • View Blog
    This option is Very Helpful in viewing the Blog Posts. It can be same as the monthly publishing book. The Posted blog can be cleared by the Administrator.
  • Type in Nepali
    You Can type words or sentenses in nepali by simply typing the corresponding words letter in english as you type sms in mobile.
